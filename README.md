# Selenium Automation Script for Amazon.in

## Overview
This Python script demonstrates automated web browsing using Selenium WebDriver with Firefox for searching products on Amazon India's website.

## Prerequisites
- Python 3.x installed
- Mozilla Firefox installed
- Selenium WebDriver for Python (`pip install selenium`)

## Setup Instructions
1. **GeckoDriver Setup:**
   - Download GeckoDriver from [Mozilla's GitHub](https://github.com/mozilla/geckodriver/releases) and place it in `C:\Users\nanda\OneDrive\Desktop\geckodriver-v0.34.0-win32\geckodriver.exe`.
   
2. **Firefox Binary Location:**
   - Ensure your Firefox binary path is correctly set in the script (`firefox_binary_path` variable).

3. **Python Environment:**
   - Install required Python packages using `pip install -r requirements.txt`.
